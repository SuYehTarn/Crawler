class UrlMatcher.UrlMatcher(scheme_pattern='http|https', domain_pattern='.*', path_pattern='.*')

Bases: object

The device to check if given url matched the specified patterns.

Attributes:

  • scheme_pattern (str): The regular expression string for matching the scheme of a given url.

  • domain_pattern (str): The regular expression string for matching the domain of a given url.

  • path_pattern (str): The regular expression string for matching the path of a given url.

Args:

  • scheme_pattern (str): The regular expression string of url scheme. For example, http, https, etc. Default is r'http|https'.

  • domain_pattern (str): The regular expression string of url domain. Default is r'.*'.

  • path_pattern (str): The regular expression string of url path. Default is r'.*'.

isIncluded(url)

The method of checking if a given url matched the specified patterns.

Args:

  • url (str): The url attempted to check.

Returns:

bool: True if the url is matched the patterns, False otherwise.
